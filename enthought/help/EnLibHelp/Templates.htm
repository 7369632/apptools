Templates

Templates are predefined data objects that can be used for creating new objects in your project. Like a template in a word-processing or spreadsheet application, a VMS template defines the initial data for the new object, but once the new object is created, changes to the object and changes to the template do not affect one another.

Templates in ProAVA2 are predefined formats for plots. You can save a plot template and apply it to other sets of data than the one it was created with.

You can use templates in VMS for:

  • Geometry models

  • Materials

  • Data flows

A geometry model template contains an entire model, but that model may consist of a functional subset of an entire mixer, such as an impeller.

A data flow template can contain an entire data flow, or a subset as small as a single step.

Templates are stored in repositories, which may be shared or personal.

Importing Templates

Applying a template to a plot is called "importing the template.

  1. In a log plot, Column Log View, Synthetic Trace View, or Simulation View window, on the Plot menu, click Restore Template. The Import Item dialog box appears.

  2. If necessary, expand the folders for the repository location for the template you want to use.

  3. Select a template in the repository pane.

  4. Click Import to apply the plot template to the current plot.

Creating an object from a template is called "importing" the template.

Importing a Geometry Model Template

  1. In the Current Project view, right-click a geometry model, point to Import, and click From Repository. The Import Item dialog box appears.

  2. If necessary, expand the folders for the repository location for the template you want to use.

  3. Select a template in the repository pane. A visualization of the template appears in the Preview pane, along with metadata information about it. (Depending on the complexity of the model, the visualization may take a few seconds to appear.)

  4. Click Import to populate the geometry model in your project with the model template.

Importing a Material Template

  1. In the Current Project view, right-click the Materials node for a test, and click Import on the shortcut menu. The Import Material dialog box appears.

  2. If necessary, expand the folder containing the template want to import, in the repository pane.

  3. Select the name of a template. Details about the template are displayed in the Preview pane.

  4. Click Import. The new material appears in the Current Project view under the Materials node.

Importing a Data Flow Template

To add a data flow template to an existing data flow, see Adding a Template to a Data Flow.

  1. In the Current Project view, right-click the Data Flow node, and click Import on the shortcut menu. If the data flow contains steps, a Confirm Import dialog box appears.

  2. If necessary, click Yes to replace your existing data flow with the imported one. The Import Data Flow dialog box appears.

  3. If necessary, expand the folder containing the template you want to import, in the repository pane.

  4. Select the name of a template. Details about the template are displayed in the Preview pane.

  5. Click Import. The steps of the data flow appear in the Current Project view under the Data Flow node.

Merging a Geometry Model Template

  1. In the Geometry Structure toolbar, click Merge from repository (). The Import Item dialog box appears.

  2. Follow the steps for Importing a Geometry Template, starting with Step 2. If there are names in the original model and in the template that are the same, the Geometry Model Merge Naming Collisions dialog box appears.

  3. For each variable that has the same name in both models, decide whether you want to rename it or map it to a name in the project's model. If you map a name, all references that name in the objects created by merging the template will be replaced by references to the mapped name in the project's model.

    • To rename an item, type a name in the Rename To? column.

    • To map an item, select a name Map To? column. The name that is selected by default is the matching name in the project's model.

  4. If necessary, click the tabs for Surface Groups, Volume Groups, and Reference Frames, and repeat Step 3 for each of those types of items.

  5. Click OK.

NOTE: It may be necessary to adjust variable values (for example, position and rotation) after merging a template into a model.

Exporting Templates

  1. In a log plot, Column Log View, Synthetic Trace View, or Simulation View window, on the Plot menu, click Save Template. An Export Item dialog box appears.

  2. Select a folder in which to save the template.

  3. Type a File name for the template.

  4. Click Export.

Exporting a Geometry Model as a Template

  1. In the Current Project view, select the node for the geometry model you want to export.

  2. Do one of the following:

    • On the main toolbar, click Export selection to a repository ().

    • Right-click the node, point to Export, and click To Repository on the shortcut menu.

    The Save Geometry Model Template dialog box appears.

  3. Select a folder in which to save the template.

  4. Type a File name for the template.

  5. Click Export.

Exporting a Material as a Template

  1. In the Current Project view, select the node for the material you want to export.

  2. Do one of the following:

    • On the main toolbar, click Export selection to a repository ().

    • Right-click the node and click Export on the shortcut menu.

    The Save Material dialog box appears.

  3. Select a folder in which to save the template.

  4. Type a File name for the template.

  5. Click Export.

Exporting a Data Flow as a Template

To export a subset of a data flow, see Exporting a Data Flow.

To export a whole data flow:

  1. In the Current Project view, select the Data Flow node.

  2. Do one of the following:

    • On the main toolbar, click Export selection to a repository ().

    • Right-click the node and click Export on the shortcut menu..

    The Save Data Flow dialog box appears.

  3. Select a folder in which to save the template.

  4. Type a File name for the template.

  5. Click Export.
