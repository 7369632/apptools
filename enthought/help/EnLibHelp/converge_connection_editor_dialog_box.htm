Converge Connection Editor Dialog Box

To access this dialog box: Double-click a chip in the Dataflow tab of a model window.

Use this dialog box to edit the names of a chip, its bundles, and its ports. You can also move ports to different bundles and hide or show output ports.

The box on the left side of the dialog shows the hierarchy of the chip, with subfolders for inputs and outputs, bundles, and ports.

  • You can expand the hierarchy and select items within it.

  • You can drag a port to a different bundle of the same type (input or output).

Name: The name of the selected item. Use this field to edit the name of the item.

Visible: This checkbox is available only for output bundles and ports. If it is selected, the item is visible in the model window tabs. If it is cleared, the item is not visible in the model window, but it is still visible in the Navigator tab.
