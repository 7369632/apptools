Edit Filter For Inputs Dialog Box

To access this dialog box:

  1. In the View list of the Inputs tab of a model window, select Customize. The Customize Filters dialog box appears.

  2. Select the Evaluation filter template, or a filter based on it, and click New or Edit.

Use this dialog box to specify criteria for a filter for the table of output ports on the Outputs tab of a model window. When a filter is applied to the table, only the input ports that match the criteria of the filter are displayed in the table.

Function name contains: A string to search for in the function (chip) name.

Input name contains: A string to search for in the input port name.

Guidance: A string to search for in the Guidance field of the input port.

Connected: Select one of the following:

  • Don't care: The connection status of the input port does not affect the search.

  • Connected

  • Not connected
