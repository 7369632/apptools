Component Library Browser Dialog Box

To access this dialog box, do one of the following:

  • On the main window toolbar, click View component library (). When the dialog box is accessed in this way, you can view components, but they are not applied anywhere.

  • In a Workflow Properties dialog box, click Add a new component (). The component you select in the Component Library Browser dialog box is added to the workflow.

  • In the ProAVA2 Explorer tab, right-click a column or layer node, point to New Property, and click Component. The component you select in the dialog box is added to the column or layer as a component property.

Use this dialog box to view and select script components.

Available Script Components: Displays names and descriptions of available components. Click a component to select it.

Inputs: Displays input parameters for the selected component.

Outputs: Displays output parameters for the selected component.

Code: Displays the code that defines the component.

You can change the width of columns in the Available Script Components, Inputs, or Output lists:

  1. Position the pointer over the edge of a column header (for example, between Name and Description). The pointer changes to a double-headed arrow.

  2. Drag the column boundary to adjust the column width.
