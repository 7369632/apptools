Projects

A project is a collection of data that are saved and used together. Project data are saved in a file whose name is the name of the project.

You can do the following operations with projects:

Task

 

Create a project.

  1. Do one of the following:

    • On the File menu, point to Project, and click New.

    • On the toolbar, click the New icon, .

    A New Project dialog box appears.

  2. Specify a Location for the project. Do one of the following:

    • To create the project in the default location, edit the last item in the Location path to specify the name of the project.

    • To create the project in a different location, do the following:

      1. Click Browse. A Browse for Folder dialog box appears.

      2. Browse to select the folder where you want to create the project, and click OK.

      3. In the Location box, add a backslash ("\"), and the name you want to give the project, to the end of the location path.

  3. Click OK. A project tree for the new project appears in the Current Project tab.

Open a project.

  1. Do one of the following:

    • On the File menu, point to Project, and click Open.

    • On the toolbar, click the Open icon, .

    An Open Project dialog box opens.

  2. Select the file that contains the project you want to open, and click Open. The project tree for the project you selected appears in the Project tab.

Save changes to a project.

Do one of the following:

  • On the File menu, point to Project, and click Save.

  • On the toolbar, click the Save icon, .

A message dialog box notifies you that the project was saved.

Save a project to a different location.

  1. Do one of the following:

    • On the File menu, point to Project, and click Save As.

    • On the toolbar, click the Save As icon, .

    A Save Project As dialog box appears.

  2. Navigate to the folder where you want to save the project.

  3. Enter the new File name for the project, and click Save.

Close an open project.

Do one of the following:

  • On the File menu, point to Project, and click Close.

  • On the toolbar, click the Close icon, .
