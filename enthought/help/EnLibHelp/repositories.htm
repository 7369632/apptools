Repositories

Repositories are locations where templates are stored. They support the following types of templates:

  • Shared: Created and maintained centrally for users in the organization.

  • Bundled: Provided with the application.

  • User-defined: Created and maintained by you, in a location to which you have access.

Repositories and the templates they contain are implemented as directories and files. Therefore, your access to them is determined by your access to the underlying file system. Typically, shared and bundled repositories are locations that you can read but not write. A user-defined repository that you create must be located in a part of the file system to which you have write access as well as read access.

You can do the following operations with repositories:

  • Creating a repository root

  • Creating a repository folder

  • Deleting a folder or template

You can work with repositories using a Repository pane in an Import Item or Export Item dialog box.

Creating a Repository Root

A repository root is the root directory for a repository.

  1. In a Repository pane, select the top-level Repository node.

  2. Click Add a new repository root () in the pane's toolbar. A New Repository Root dialog box appears.

  3. Enter a user-friendly Name for the repository.

  4. Enter a Path to the directory that will be the repository root, or click Browse to open a Select a Directory dialog box, so you can select a directory.

  5. Click OK in the New Repository Root dialog box. The new repository root appears as a child folder of the top-level Repository node.

Creating a Repository Folder

  1. In a Repository pane, select a repository root or folder to which you have write access.

  2. Click Create a new repository folder () in the pane's toolbar. A New Folder dialog box appears.

  3. Enter a user-friendly Name for the folder, and click OK. The new folder appears as a child of the folder you originally selected.

Deleting a Template or Folder

  1. In a Repository pane, select a folder or template to which you have write access.

  2. Click Delete the selection () in the pane's toolbar. A confirmation dialog box appears.

  3. Click Yes to delete the item.
