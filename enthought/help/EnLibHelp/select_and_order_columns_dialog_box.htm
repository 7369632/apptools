Select and Order Columns Dialog Box

To access this dialog box: Click the Set user preferences for table icon in the Inputs tab or Outputs tab of a script window.

Use this dialog box to choose columns to display in a script window Inputs tab or Outputs tab, and to specify their order.

The list on the left side of the dialog box contains names of columns that are not currently displayed in the tab.

The list on the right side contains names of columns that are currently displayed, in the order that they are displayed.

Use the buttons to move names between the list, and to reorder the right-hand list.

>: Move the name that is selected in the left-hand list to the position above the selection in the right-hand list.

<: Move the name that is selected in the right-hand list to the position above the selection in the left-hand list.

Move up: Move the name that is selected in the right-hand list up by one position. The corresponding column in the table moves to the left by one position.

Move down: Move the name that is selected in the right-hand list down by one position. The corresponding column in the table moves to the right by one position.

As names are added, removed, or reordered in the right-hand list, the columns in the table change immediately.
