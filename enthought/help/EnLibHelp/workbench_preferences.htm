Workbench Preferences

To access this page:

  1. On the Tools menu, click Preferences. The Preferences dialog box appears.

  2. In the list on the left side of the dialog box, select Workbench.

Use this preference page to set preferences for the operation of the application.

Prompt on exit: If checked, the application asks for confirmation that you want to exit when you give an exit command.

Show tool names: If checked, labels are displayed for buttons on the main toolbar.

Default perspective: Select the name of the perspective that you want to be displayed when you start the application. Last one used indicates that the perspective you were using at the end of your previous session will be restored.
