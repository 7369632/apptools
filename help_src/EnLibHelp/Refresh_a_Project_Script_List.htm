Refreshing a Script List

The script nodes in the ProAVA2 Explorer tab represent script files located in the file system. The Global Scripts node lists script files in the directory root\apps\cplab\scripts, where root is the directory of the ProAVA2 installation. The Scripts nodes for a particular project lists the files in the scripts subdirectory of the project directory.

If you add or remove scripts in one of these directories from outside of ProAVA2, you might need to refresh the list of scripts for the node in the ProAVA2 Explorer tab.

  • Right-click the Global Script node or the Scripts node of a project in the ProAVA2 Explorer tab, and click Refresh on the shortcut menu. The children of the node are updated to reflect the current contents of the corresponding scripts directory.
