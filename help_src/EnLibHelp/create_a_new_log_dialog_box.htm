Create a New Log Dialog Box

To access this dialog box: In the Inputs tab of a model window, select an input port that represents a log, and click Create Log in the bottom panel.

Use this dialog box to create a new constant log, which will be used as the value of an input port.

Index: The index log to use for the new log. Drag an index log from the Navigator tab onto this field. The name of the index log appears in this field, and the name of the log suite that the index belongs to appears for Log suite. This field may be pre-filled, if you dragged a depth index to the Depth Index field on the Inputs tab of the model window.

Log suite: The log suite that the new log will be added to. This field is filled in when you drop an index log on the Index field.

Log name: The name for the new log.

Value: The constant value to use for the new log.

Units: The units of the new log. The options for this field depend on the units required by the input port.

Family: The unit family of the new log. This field is filled based on the units required by the input port.

Units system: The units system designated for the port. This information is used by the program when converting data between connected ports.
