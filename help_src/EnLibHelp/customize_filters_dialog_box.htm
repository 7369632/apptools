Customize Filters Dialog Box

To access this dialog box: In the View list of the Inputs tab or Outputs tab of a model window, select Customize.

To access this dialog box: In the View list of the Geometry Variables or Selected Geometry Variables view, select Customize.

Use this dialog box to create, edit, and delete filters for a table of items.

The list on the left side of the dialog box contains names of filters that can be applied to the table. Select a filter in the list, and then click one of the buttons on the right to operate on it.

Edit: Displays the Edit filter dialog box so you can modify the selected filter.

New: Creates a new filter based on the selected filter, and displays the Edit filter dialog box so you can modify the new filter.

Apply: Applies the selected filter to the table. Only items that meet the filter criteria are displayed in the table.

Delete: Deletes the selected filter.

NOTE: You cannot edit or delete filter templates.
