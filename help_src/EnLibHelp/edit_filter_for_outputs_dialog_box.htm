Edit Filter For Outputs Dialog Box

To access this dialog box:

  1. In the View list of the Outputs tab of a model window, select Customize. The Customize Filters dialog box appears.

  2. Select a filter and click New or Edit.

Use this dialog box to specify criteria for a filter for the table of output ports on the Outputs tab of a model window. When a filter is applied to the table, only the output ports that match the criteria of the filter are displayed in the table.

Function name contains: A string to search for in the function (chip) name.

Output name contains: A string to search for in the output port name.
