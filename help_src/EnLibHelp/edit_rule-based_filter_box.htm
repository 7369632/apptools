Edit Rule-Based Filter Dialog Box

To access this dialog box:

  1. In the View list of the Inputs tab of a model window, select Customize. The Customize Filters dialog box appears.

  2. Select the Rule-based filter template, or a filter based on it, and click New or Edit.

To access this dialog box:

  1. In the View list of the Geometry Variables view, select Customize. The Customize Filters dialog box appears.

  2. Select the Rule-based filter template, or a filter based on it, and click New or Edit.

Use this dialog box to specify criteria for a filter for a table. When a filter is applied to the table, only the items that match the criteria of the filter are displayed in the table.

Filter name: The name of the filter. This name is dynamically constructed, based on the rules specified, unless you explicitly type a new name.

The dialog box contains a table of rules, which may be empty. The table contains the following columns:

or: Indicates whether the rule should be conjoined ("ANDed") or disjoined ("ORed") with other rules in the table.

Name: An attribute on the item being filtered.

Operation: A Boolean operation relating the Name to the Value.

Value: A value to be related to the Name.

To add a rule to the table, click the Insert new item () icon, and then modify the cells of the row to define the rule.

To delete a rule, select it and click the Delete current item () icon.
