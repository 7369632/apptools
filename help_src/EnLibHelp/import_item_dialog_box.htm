Import Item Dialog Box

To access this dialog box, do one of the following:

  • In the Current Project view, right click a geometry model node, point to Import, and click From Repository on the shortcut menu.

  • In the Geometry Structure toolbar, click Merge from repository ().

  • In a log plot, Column View, Synthetic Trace View, or Simulation View window, on the Plot menu, click Restore Template.

Use this dialog box to preview geometry templates, and select and import one into your model.

Use this dialog box to select and import plot templates.

This dialog box contains two panes: Preview and Repository.

Preview Pane

The Preview pane displays a visualization of the geometry template representation of the item that is selected in the Repository pane.

Repository Pane

The repository pane displays a hierarchy of repository folders and templates.

Select a template to display it in the Preview pane.

 

Click Import to import the selected template.
