Logger Preferences

To access this page:

  1. On the Tools menu, click Preferences. The Preferences dialog box appears.

  2. In the list on the left side of the dialog box, select Logger.

Use this preference page to set preferences for the operation of the Logger view and the Quality Agent dialog box.

Logger Settings

Level: Select the level of messages to be logged. All messages at the selected level and higher levels of severity will be logged.

Quality Agent Settings

Enable quality agent: If checked, the Quality Agent dialog box can be opened from the Logger view.

SMTP server: The mail server to use to send feedback e-mail messages.

From address: Default address to use in the From: header of feedback e-mail messages.

To address: Default address to use in the To: header of feedback e-mail messages.
