Model Window

To access this window: Open a model.

The model window contains the following tabs:

  • Dataflow

  • Inputs

  • Outputs

  • Optimization

Dataflow Tab

The Dataflow tab contains a toolbar and a "canvas" or work area for arranging and connecting chips for a model.

Related Topics:

Inputs Tab

The Inputs tab contains a table that lists input ports of the chips currently in the model. You can apply filters to restrict the list of input ports displayed.

If the input port that is selected in the table requires a well log, a panel appears in at the bottom of the window, showing the name of the log (if any) that is associated with the input port. You can drag a well log from the Navigator tab onto the Log name field, or click Create Log to create one on the spot.

Related Topics:

Outputs Tab

The Outputs tab contains a table that lists output ports of the chips currently in the model. You can apply filters to restrict the list of output port displayed.
