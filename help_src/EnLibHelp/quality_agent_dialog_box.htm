Quality Agent Dialog Box

To access this dialog box: Double-click an entry in the Logger view.

You must enable this dialog box in Logger Preferences. Otherwise, a Log Message Detail window appears instead.

Use this dialog box to send feedback to the application developers.

Message: Contains the full text of the Logger entry.

Report Information

Subject: Type a summary of your message.

To: The address to send the message to. The default address sends it to the issue-tracking system for CP Lab.

Cc: Type an e-mail address for anyone else you wish to notify.

From: Type your own e-mail address.

SMTP Server: Enter the name of a mail server for your organization.

How critical is this issue? Select one of the following options:

  • Low: A minor problem, or one that has an easy work-around.

  • Medium: A problem where the program is not working as intended or expected.

  • High: A problem that significantly interferes with your ability to use the application, and has no work-around.

  • Critical: A problem that prevents you from using the application, and has no work-around.

Additional Comments: Type your comments about the program.

Include Workspace File: Select this to attach a copy of your current project to the e-mail message. If an error depends in some way on your specific data, attaching the project file helps developers reproduce the problem.

Send: Click to send an e-mail message.

Cancel: Click to close the dialog box without sending an e-mail message.

NOTE: The program sends an e-mail message only if your computer is connected to the Internet, and has access to the SMTP Server you select.
