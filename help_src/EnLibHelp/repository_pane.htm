Repository Browser Pane

This pane appears in the Data Flow perspective, and in Import Item and Export Item dialog boxes.

Use this pane to work with the hierarchy of template repositories.

Toolbar

Icon

Operation

Description

Add a new repository root

Displays a New Repository Root dialog box, so you can define a repository root. This button is available only when the top-level Repository node is selected.

Create a new repository folder

Displays a New Folder dialog box, so you can define a new folder in a repository. This button is available only when a folder or root is selected to which you have write access.

Delete the selection

Displays a confirmation dialog to confirm that you want to delete the selected item. This button is available only when a root, folder, or template is selected to which you have write access.
