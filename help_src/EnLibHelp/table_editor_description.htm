A table contains one or more columns and zero or more rows. You can set different values for each cell in the table. Some cells are text boxes; others may contain a drop-down list of items you can select.

You can sort a table based on the values in a column by clicking the label for that column. Each successive click sorts the table in the following order:

  1. Ascending order by the column's values.

  2. Descending order by the column's values.

  3. Not sorted by the column.

The toolbar of a table may contain the following buttons:

Description
