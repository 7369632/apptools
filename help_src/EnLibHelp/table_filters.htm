Table Filters

For tables that have a View list in the toolbar, you can create and modify table filters, to restrict the set of items displayed in the table.

Applying a Filter

  • In the View list, select the filter name.

The filter criteria are applied to the table, so that only items matching those criteria are displayed.

Removing a Filter

  • In the View list, select No filter.

All items are displayed in the table.

Creating a Filter

  1. In the View list, select Customize. The Customize Filters dialog box appears.

  2. Select a filter template or an existing filter. Filter templates have "template" in the name.

  3. Click New. The Edit Filter dialog box appears. The contents of this dialog box varies depending on the type of template the new filter is based on. If the new filter is based on another filter, the dialog box contents depend on the template on which the existing filter was based.

  4. Specify filter criteria and click OK. See  Filter Templates for details.

  5. Follow the steps in Testing a Filter and Editing a Filter, as necessary, until you are satisfied with the results of the filter.

  6. Click OK to close the Customize Filters dialog box.

Testing a Filter

  1. If necessary, select Customize in the View list, and select the filter in the Customize Filters dialog box.

  2. Click Apply.

  3. Review the table to determine whether the appropriate items are displayed.

  4. If the results are satisfactory, click OK. Otherwise, proceed with Step 2 of Editing a Filter.

Editing a Filter

  1. If necessary, select Customize in the View list, and select the filter in the Customize Filters dialog box.

  2. Click Edit. The Edit Filter dialog box appears.

  3. Modify filter criteria and click OK. See  Filter Templates for details.

  4. Click OK to close the Customize Filters dialog box.

Deleting a Filter

  1. If necessary, select Customize in the View list, and select the filter in the Customize Filters dialog box.

  2. Click Delete. The Delete button is not available if a template is selected. The filter is removed from the list.

  3. Click OK to close the Customize Filters dialog box.
