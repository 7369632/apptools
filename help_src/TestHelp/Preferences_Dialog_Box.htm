Preferences Dialog Box

This topic describes the Envisage Preferences dialog box.

 

The dialog box has two panes:

  • A tree pane on the left

  • A data pane on the right

 

The tree pane displays a hierarchy of page names. The data pane displays the contents of the page that is currently selected in the tree pane.

 

To change preference settings:

  1. In the tree pane, click the name of the page containing the setting you want to change.

  2. Make your changes in the corresponding data pane.

  3. Repeat steps 1 and 2 for any other pages whose settings you want to modify.

  4. Click Done when you are finished making changes.

 

Refer to the help for each plug-in to learn about its preference settings.
